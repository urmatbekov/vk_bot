Зайдите в папку проекта и создайте виртуальную среду с командой
virtualenv -p python3 envbot
И активируйте с командой
source env/bot/bin/activate
и запустите этот скрипт в терминале
pip install -r requirements.txt
a потом откройте файл config.py и пропишите свой токен(токен возьмете c этого бота "@BotFather")
а потом запустите файл bot.py с командой
python bot.py

    Надо спрашивать даты таком формате "20 августа"
